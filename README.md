# the-construct
The Construct. Our loading program. We can load anything from Email, to instant messaging, document / office suite, document storage, business administration, user administration (IAM). 

Anything we need.

The Construct is what your tools would look like if they were built for you — not for the company that sells them.

## The Tech Stack

* Keycloak SSO
* MinIO Storage
* OnlyOffice (Document collaboration services)
* Quarkus
* Stalwart Mail - Open Source edition
* Mattermost

All available for on-prem or cloud infrastructure via K8s.
